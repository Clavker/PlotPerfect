# AutoCAD Tasks - Автоматизация задач AutoCAD

Программа для автоматизации рутинных операций в AutoCAD. Позволяет выполнять задачи с графическим интерфейсом, отслеживать статус выполнения и получать подробные отчеты.

## 📋 Возможности

- 🎨 **Графический интерфейс** на Tkinter
- 📊 **Отслеживание статуса** выполнения задач
- 📝 **Детальное логирование** в реальном времени
- ✅ **Подробные отчеты** по каждому листу
- 🔄 **Автоматическая обработка** всех листов чертежа

## 🎯 Текущие задачи

### Задача 1: Выставить рамки для печати
- Отправляет номер листа (координаты 1.06, 3.51) на задний план
- Выводит рамку чертежа (координаты 8, 5) на передний план
- Автоматически обрабатывает все листы в чертеже
- Выводит подробный отчет о результатах

## 🚀 Установка и запуск

### Требования
- Python 3.8 или выше
- AutoCAD (любая версия с поддержкой COM)
- Poetry (менеджер зависимостей)

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Clavker/autocad-tasks.git
cd autocad-tasks
Установите зависимости:

bash
poetry install
Запустите программу:

bash
poetry run python src/autocad_tasks/main.py
📖 Использование
Запустите AutoCAD и откройте нужный чертеж

Запустите программу из терминала

В открывшемся окне выберите задачу для выполнения

Нажмите кнопку "Запустить" или "AUTO" для выполнения всех выбранных задач

Следите за процессом в окне лога

Получите подробный отчет о выполнении
